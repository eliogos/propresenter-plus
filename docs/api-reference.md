# API Reference

The library provides several sub-modules to interact with different parts of ProPresenter.

## Slide
- `pro.slide.getCurrent()`: Get status of the active slide.
- `pro.slide.getIndex()`: Get the index of the current slide.

## Presentation
- `pro.presentation.triggerNext()`: Trigger the next slide.
- `pro.presentation.triggerPrev()`: Trigger the previous slide.
- `pro.presentation.getActive()`: Get the active presentation details.

## Timers
- `pro.timers.startAll()`: Start all timers.
- `pro.timers.stopAll()`: Stop all timers.
